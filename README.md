# 🏦 ATBank AI Customer Analytics

AI-powered banking analytics platform for **customer churn prediction, automatic data insights, and customer segmentation**.

This project demonstrates how data science and machine learning can be used to analyze customer behavior and predict potential churn using an interactive analytics dashboard.

Built using **Python, Streamlit, Machine Learning, and Data Visualization**.

---

# 🚀 Features

### 📊 Automatic Data Insights

The system automatically analyzes uploaded datasets and generates insights such as:

- Outlier detection
- Data distribution analysis
- Skewness detection
- Missing value analysis
- Correlation insights

Example insights generated by the system:

• account_length has 18 unusually high values
• number_vmail_messages has 45 unusually high values and shows right-skewed distribution
• total_day_minutes has 18 unusually high values and shows right-skewed distribution
• total_day_calls has 16 unusually high values
• total_eve_minutes has 18 unusually high values
• total_night_minutes has 20 unusually high values and shows right-skewed distribution
• total_intl_calls has 35 unusually high values and shows right-skewed distribution
• number_customer_service_calls has 21 unusually high values and shows right-skewed distribution

These insights help analysts quickly identify **abnormal behavior patterns in customer data**.

---

### 🧠 AI Churn Prediction

The platform integrates a **RandomForest machine learning model** to predict customer churn.

Capabilities:

- Real-time prediction
- ONNX model inference
- Feature-based churn risk detection

---

### 👥 Customer Segmentation

Customer groups are discovered automatically using **K-Means clustering**.

Benefits:

- Identify high-value customers
- Detect risky customer segments
- Discover hidden behavioral patterns

---

### 📈 Interactive Data Dashboard

The system provides interactive visualizations including:

- Feature distribution
- Missing value charts
- Correlation analysis
- Dataset statistics
- Customer usage patterns

All charts are powered by **Plotly interactive visualization**.

---

### 🔐 User Authentication System

The platform includes a full authentication workflow:

- User registration
- Email verification
- Password reset via email
- Secure login system

# 🛠 Installation

Clone the repository

git clone https://github.com/honganhuit/Bank-AI-Customer-Analytics.git

cd Bank-AI-Customer-Analytics

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py

---

# ☁️ Deploy with Streamlit

1. Push the project to GitHub
2. Open Streamlit Cloud
3. Connect the repository
4. Deploy the file `app.py`

The dashboard will be available online instantly.
