# DATA DICTIONARY

| Field Name        | Type   | Description                 |
| ----------------- | ------ | --------------------------- |
| CustomerId        | int    | Unique customer ID          |
| Age               | int    | Customer age                |
| Balance           | float  | Account balance             |
| Tenure            | int    | Years with bank             |
| NumOfProducts     | int    | Number of products          |
| IsActive          | int    | Active status               |
| Salary            | float  | Estimated salary            |
| Churn Probability | float  | Predicted churn probability |
| Risk Level        | string | High/Medium/Low             |
| Cluster           | int    | Customer segment            |
