# BUSINESS RULES

## Authentication

- Only registered users can access system
- Password must contain letters and numbers
- OTP required for password reset

---

## Data Validation

- Only CSV files allowed
- Dataset must contain required features
- Missing and duplicate data must be detected

---

## Data Analysis

- Only numeric data is used
- Correlation threshold > 0.8

---

## Segmentation

- Minimum 2 features required
- Only numeric features used

---

## Churn Prediction

- High Risk: > 0.7
- Medium Risk: 0.4 – 0.7
- Low Risk: < 0.4

---

## Output

- Results must be displayed clearly
- Users can download results
- Errors must be shown for invalid inputs
